# PROJET PROG5 2024-2025 Editeur de liens

## Pour compiler

La première fois

```sh
./fresh_start.sh && make
```

Ensuite

```sh
make
```

## Pour lancer les tests directement

La première fois

```sh
./fresh_start.sh && make test
```

Ensuite

```sh
make test
```

Pour plus de détails

```sh
./run_tests.sh -h
```
